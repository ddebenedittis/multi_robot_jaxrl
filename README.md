# Multi-Robot Reinforcement Learning with JAX

Motion planning of mobile robots with obstacle avoidance.

## Installation

Create a virtual environment and install the dependencies:
```shell
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```
## Usage

Run the [`cluttered_env.ipynb`](./multi_robot_jaxrl/cluttered_env.ipynb) notebook.

Result: robot in white, moving obstacles in blue, and goal in green:
